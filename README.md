## Описание
Простой аналог игры Flappy Bird, использующий средства OpenTK 3.2.0 на платформе .NET Framework.<br>

Проект условно можно поделить на 3 части:
* игровой движок (Engine);
* игровые объекты (Game);
* описание окна приложения (Program).
